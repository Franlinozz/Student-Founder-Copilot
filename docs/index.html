Student Founder Copilot

A calm, professional Telegram agent that helps real students verify, plan, and ship hackathon submissions in minutes — built for the College.xyz track at The Synthesis.

What it handles

  • 🪪 Verification intake Secure flow that captures name, school email, graduation year, and ID photo — then hashes + anchors proof on Base.
  • 🧭 Opportunity scout Curated JSON feed of hackathons, bounties, and internships students can filter inside Telegram.
  • 🧠 Idea → pack Prompt templates that turn rough ideas into submission-ready briefs with problem, solution, milestones, and hash.
  • ⛓️ Base anchoring CLI + bot command to anchor verification hashes (tx 0xbda3987…539c0 for Nonso’s proof).
